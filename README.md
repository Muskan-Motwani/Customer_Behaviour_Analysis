# 🛍️ Customer Shopping Behavior Analysis

An end-to-end Data Analytics project that explores customer shopping behavior using **Python, PostgreSQL, and Power BI**. This project demonstrates the complete analytics workflow—from data cleaning and preprocessing to SQL-based business analysis and interactive dashboard development.

---

## 📌 Project Overview

The objective of this project is to analyze customer purchasing behavior and uncover actionable business insights that can help improve marketing strategies, customer retention, and overall business performance.

The project follows a complete data analytics pipeline:

- Data Cleaning & Preprocessing using Python
- Data Storage & Business Analysis using PostgreSQL
- Interactive Dashboard Creation using Power BI
- Business Recommendations based on insights

---

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- PostgreSQL
- SQL
- Power BI
- Jupyter Notebook

---

## 📂 Dataset Information

- **Rows:** 3,900
- **Columns:** 18

The dataset contains:

- Customer Demographics
- Product Categories
- Purchase Amount
- Subscription Status
- Review Ratings
- Shipping Type
- Discounts & Promotions
- Previous Purchases
- Shopping Frequency

---

## 📊 Project Workflow

### 1. Data Cleaning (Python)

- Loaded dataset using Pandas
- Handled missing values
- Standardized column names
- Created Age Groups
- Engineered purchase frequency features
- Removed redundant columns
- Loaded cleaned dataset into PostgreSQL

---

### 2. SQL Business Analysis

Performed SQL queries to answer important business questions including:

- Revenue by Gender
- Top Rated Products
- Customer Segmentation
- Subscriber vs Non-Subscriber Analysis
- Shipping Type Comparison
- Revenue by Age Group
- Most Discount-Dependent Products
- Top Products by Category
- Repeat Buyer Analysis

---

### 3. Power BI Dashboard

Developed an interactive dashboard featuring:

- KPI Cards
- Revenue Analysis
- Customer Segmentation
- Product Performance
- Subscription Analysis
- Purchase Trends
- Interactive Filters (Slicers)

---

## 📈 Key Insights

- Subscription members generated higher overall revenue.
- Loyal customers contributed significantly to total sales.
- Certain products relied heavily on discounts for purchases.
- High-rated products consistently achieved better sales performance.
- Age groups showed different purchasing behaviors.
- Express shipping customers generally spent more per purchase.

---

## 💡 Business Recommendations

- Introduce attractive subscription benefits.
- Strengthen customer loyalty programs.
- Optimize discount strategies.
- Promote top-rated products.
- Target marketing campaigns based on customer segments and age groups.
